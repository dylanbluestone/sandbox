# Dev Sandbox

A personal feature lab built on Next.js 14. Each feature is developed with production-grade quality gates — ESLint, Prettier, Husky pre-commit hooks, and GitHub Actions CI — so experiments stay clean and tested.

→ See [PLAYGROUND.md](./PLAYGROUND.md) for full documentation.

## Quick start

```bash
npm install
npm run dev       # http://localhost:3000
```

## Feature demos

| Feature  | Route                                                     | Description                                                                                |
| -------- | --------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| Age Gate | [`/demos/age-gate`](http://localhost:3000/demos/age-gate) | Configurable age verification form — 5 live configurations, props table, and code examples |

## Commands

```bash
npm run dev          # Start dev server
npm test             # Run all tests
npm run lint         # ESLint
npm run format       # Prettier
npm run build        # Production build
```
